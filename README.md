este é um bot de música para discord desenvolvido inteiramente em Python.

Suas funcionalidades:
- ele consegue entrar em canais de voz
- possui todas as funções de um BOT de musica para discord
- toca musicas de acordo com oque voce enviar
- responde comandos personalizados

comandos:
1. !ping / responde com "Pong! 🏓"
2. !oi / diz olá
3. !dado / rola um dado
4. !entrar / entra no canal de voz
5. !tocar <pesquisa> / toca uma música
6. !fim / finaliza a música atual
7. !sair / sai do canal de voz
8. !ajuda / mostra os comandos disponíveis
9. !continuar / retoma a música pausada
10. !pausar / pausa a música em execução
11. !encerrar / limpa a fila de músicas

Suas Tecnologias:
- Python
- discord.py
- FFmpeg

Como instalar:
1. clone o seu projeto
2. instale as dependências:

pip install -r requirements.txt

após isso:

3. configure o token do Discord
4. e agora o execute

python main.py
